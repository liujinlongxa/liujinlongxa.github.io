date: 2016-09-12 23:17:44
---

使用代码删除特定约束的正确姿势：

```objectivec
if (IOS8_OR_LATER) {
    // iOS8之后删除约束的方法
    self.authorLabelRightConstraint.active = NO;
}
else {
    // iOS8之前删除约束的方法
    [self removeConstraint:self.authorLabelRightConstraint];
}
```

注意删除约束的方法在iOS8之前和之后是不一样的。
