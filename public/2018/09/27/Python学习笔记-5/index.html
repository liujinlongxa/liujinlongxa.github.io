闭包

Python 中一切皆对象

  • 函数也是对象
  • 可以把一个函数当成另外一个参数，传递到另外的函数里，也可以把一个函数当做另一个函数的返回值

什么是闭包

  • 闭包：函数 + 上下文（环境变量）
  • 闭包的信息保存在函数的__closure__里

使用闭包注意事项

  • 如果没有捕获变量，则closure为空

闭包的应用

  • 闭包可以封装细节
